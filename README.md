# Karan Bothra - Personal Portfolio

A modern, responsive, and high-performance portfolio website built with **Next.js 15 (App Router)**, **Tailwind CSS**, and **Framer Motion**. This site serves as a digital showcase of my work, skills, and professional journey as a Software Engineer.

## üöÄ Tech Stack

- **Framework:** [Next.js 15](https://nextjs.org/) (App Router)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/)
- **Components:** Custom UI components + [Lucide React](https://lucide.dev/) & [React Icons](https://react-icons.github.io/react-icons/)
- **Theming:** [next-themes](https://github.com/pacocoursey/next-themes) for Dark/Light mode support
- **Deployment:** Optimized for Vercel/Netlify

## ‚ú® Features

- **Modern Design:** Clean, minimalist aesthetic with glassmorphism and neumorphism touches.
- **Dark/Light Mode:** Seamless theme switching with system preference detection.
- **Responsive UI:** Fully optimized for all devices (Mobile, Tablet, Desktop).
- **Scroll Animations:** Smooth element entrances and scroll-spy navigation.
- **Performance:** Highly optimized with Next.js server components and code splitting.
- **Dynamic Content:** Centralized configuration file (`data/resume.ts`) for easy updates.

## üõ†Ô∏è Getting Started

To run this project locally on your machine:

1.  **Clone the repository:**

    ```bash
    git clone https://github.com/karanbothra12/karan-personal-site.git
    cd karan-personal-site
    ```

2.  **Install dependencies:**

    ```bash
    npm install
    ```

    ```bash
    npm run dev
    ```
